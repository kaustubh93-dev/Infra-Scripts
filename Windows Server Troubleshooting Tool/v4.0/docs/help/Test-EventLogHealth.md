# Test-EventLogHealth

## SYNOPSIS
Analyzes Windows Event Logs for issues

## SYNTAX

```
Test-EventLogHealth [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Scans System and Application logs for recent errors, groups by EventID,
and checks log capacity

## EXAMPLES

### EXAMPLE 1
```
Test-EventLogHealth
```
