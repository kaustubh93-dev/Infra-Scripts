# Export-SystemReport

## SYNOPSIS
Generates a comprehensive system diagnostic report

## SYNTAX

```
Export-SystemReport [<CommonParameters>]
```

## DESCRIPTION
Creates a detailed text report with system information, resource usage,
and configuration details

## EXAMPLES

### EXAMPLE 1
```
Export-SystemReport
```
